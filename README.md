Un'applicazione Android nativa sviluppata in Java tramite Android Studio progettata per il controllo immediato ed essenziale di un termostato intelligente collegato alla rete Wi-Fi. 

L'interfaccia è pensata per essere estremamente semplice, pulita e focalizzata sulle funzioni principali senza distrazioni.

Funzionalità

1 Visualizzazione Temperatura: Mostra in tempo reale la temperatura attuale rilevata dal termostato sulla schermata principale.
2 Controllo Semplificato: Permette esclusivamente di aumentare o abbassare la temperatura desiderata tramite due comandi intuitivi (es. pulsanti `+` e `-`).
3 Connessione Wi-Fi: Comunica in modo diretto e bidirezionale con il dispositivo termostato presente nella rete locale.
